# iOS Web Simulator

A virtual iOS operating system built with React.js, Tailwind CSS, and Framer Motion.

## Features
- Realistic iOS interface
- Dynamic Island with animations
- 4 functional mini-apps (Notes, AI Chat, Portfolio, Calculator)
- Swipe gestures to close apps
- Glassmorphism effects
- PWA support
